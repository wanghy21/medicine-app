# 快速开始 - 图形化操作

## 不懂命令行？双击这些文件即可！

### 📦 第一步：安装依赖

**双击运行**：`安装依赖.bat`

这会自动下载所有需要的库文件。

**注意事项**：
- 可能需要 3-5 分钟（取决于网速）
- 如果失败，双击重试几次
- 确保 Node.js 已安装（见下方"安装 Node.js"）

### 🚀 第二步：启动应用

**双击运行**：`启动服务器.bat`

这会启动开发服务器。

**注意事项**：
- 等待看到二维码出现
- 不要关闭这个窗口
- 如果想停止，按 `Ctrl + C`

### 📱 第三步：在手机上查看

1. **安装 Expo Go**
   - 打开手机的应用商店（Google Play）
   - 搜索 "Expo Go"
   - 点击安装

2. **扫描二维码**
   - 打开 Expo Go 应用
   - 点击 "Scan QR Code"（扫描二维码）
   - 对准电脑屏幕上的二维码扫描

3. **或者手动输入**
   - 找到命令行中显示的地址，例如：`exp://192.168.1.100:8081`
   - 在 Expo Go 中点击 "Enter URL manually"
   - 输入该地址

## 🔧 安装 Node.js

如果双击 `安装依赖.bat` 时提示找不到 `npm`，需要先安装 Node.js。

### 下载安装：

1. 访问：https://nodejs.org/
2. 下载 LTS 版本（推荐）
3. 双击下载的安装包
4. 一路点击"下一步"完成安装
5. 重新双击 `安装依赖.bat`

### 验证安装：

打开新的命令行，输入：
```bash
node --version
npm --version
```

如果显示版本号，说明安装成功。

## ⚠️ 常见问题

### Q: 双击后窗口一闪而过

**原因**：可能是安装失败了

**解决**：
1. 右键点击 `.bat` 文件
2. 选择"以管理员身份运行"

### Q: 下载很慢或卡住

**解决**：
1. 多试几次
2. 检查网络连接
3. 使用手机热点试试

### Q: 手机扫描不了二维码

**解决**：
1. 确保手机和电脑在同一个 WiFi
2. 手动输入命令行中显示的地址
3. 检查电脑的防火墙设置

### Q: 手机上显示"连接失败"

**解决**：
1. 确保开发服务器窗口还在运行
2. 重启 Expo Go 应用
3. 刷新二维码（按 R 键）

## 💡 使用提示

### 停止服务器

在运行 `启动服务器.bat` 的窗口中，按 `Ctrl + C` 即可停止。

### 重新加载应用

在手机上打开 Expo Go，摇一摇手机（或右滑）
- 点击 "Reload" 重新加载
- 点击 "Debug" 查看调试信息

### 查看日志

运行 `启动服务器.bat` 的窗口会显示所有操作日志。
