# 🚀 三步开始使用

## 新手必读：就这么简单！

### 步骤 1️⃣：检查环境

**双击运行**：`环境检查.bat`

这会检查：
- Node.js 是否已安装
- npm 是否可用
- 项目文件是否完整
- 依赖是否已安装

**如果检查通过** → 进入步骤2
**如果提示需要安装 Node.js** → 先安装 Node.js

---

### 步骤 2️⃣：安装依赖

**双击运行**：`安装依赖.bat`

这会：
- 自动下载所有需要的库
- 等待 3-5 分钟（取决于网速）
- 完成后提示"安装完成"

**如果失败** → 重试几次或检查网络

---

### 步骤 3️⃣：启动应用

**双击运行**：`启动服务器.bat`

这会：
- 启动开发服务器
- 显示一个二维码
- 显示一个地址（例如：exp://192.168.1.100:8081）

**然后在手机上**：
1. 安装 Expo Go（应用商店搜索下载）
2. 打开 Expo Go
3. 扫描二维码 或 手动输入地址
4. 应用自动加载到手机上

---

## 📱 手机上安装 Expo Go

### 安装方法

**安卓手机**：
1. 打开 Google Play 商店
2. 搜索 "Expo Go"
3. 点击安装

**无法访问 Google Play**：
1. 下载 APK：https://expo.dev/go
2. 在手机上下载并安装

### 使用方法

1. 启动后选择 "Scan QR Code"
2. 对准电脑屏幕上的二维码
3. 等待应用自动加载

## ⚙️ 高级：使用命令行

如果你想用命令行操作：


## 🔧 遇到问题？

### 问题1：双击 `.bat` 文件窗口一闪而过

**解决**：右键点击 → 以管理员身份运行

### 问题2：提示找不到 `npm` 命令

**解决**：需要先安装 Node.js
1. 访问：https://nodejs.org/
2. 下载 LTS 版本
3. 双击安装
4. 重新双击 `环境检查.bat`

### 问题3：安装依赖很慢

**解决**：多试几次，或者使用手机热点

### 问题4：手机扫描不了二维码

**解决**：手动输入地址
- 找到命令行中显示的地址，例如：`exp://192.168.1.100:8081`
- 在 Expo Go 中点击 "Enter URL manually"
- 输入该地址

### 问题5：应用加载失败

**解决**：
1. 确保电脑和手机在同一 WiFi
2. 重启 Expo Go 应用
3. 在开发服务器窗口按 `R` 刷新
